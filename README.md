Диплом был на тему "Интеллектуальная система анализа и консультации тревожных расстройств" и был оценен комиссией на Отдично.

Тут представлены сам диплом и первая сыпая реализация веб сайта (к сожалению, только это нашла в чате с научным руководителем).
Сама идея диплома и реализация не такая уж "интелектуальная", так как она была основана на простой рекомендательной системе на нечеткой логике. Нечёткая логика (fuzzy logic) - используется в рекомендательных системах для учёта неопределённости и размытых данных, которые характерны для предпочтений пользователей и близости объектов, а в моем случае определения близости пользователя сайта к определенной группе.
